jquery-example

Quickly create html inline examples

Basic usage

Simply add a example-data-option to your markup.

For "example", the following code...

<h4>Markup of a div-element</h4>
<!-- declare an example markup block by using the data-example-attribute -->
<div id="div" data-example="example1">Hello, I'm a div</div>
      
<h4>Styling a div-element using an inline style block:</h4>
<!-- declare an example style block by using the data-example-attribute -->
<style data-example="example1">
  /* style declaration */
  #div {
    background: lightgray;
    border: 1px solid gray;
    border-radius: 5px;
    font-size: 16px;
    padding: 9.5px;
    margin: 0 0 10px;
  }   
</style>
<h4>Styling a div-element using jquery:</h4>
<!-- declare an example script block by using the data-example-attribute -->
<script data-example="example1" type="text/javascript">
  // set font-color via jquery
  $(function() {
    $("#div").css({
      'color': 'blue'
    });
  });        
</script>
      
<h4>Demo:</h4>
<!-- the working example will be added here -->
<div data-example="example1"></div>
